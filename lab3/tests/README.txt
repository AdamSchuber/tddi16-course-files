Diverse intressanta testfall.

Testerna kan köras med följande kommando:

cat words/swedish.txt tests/swedish1.txt | program

Här använder vi testfallet som finns i filen "swedish1.txt" tillsammans med ordlistan
"swedish.txt". Testfallen är namngivna efter vilken ordlista de vill köras tillsammans med, så
filerna "swedish*.txt" ska köras med "swedish.txt" och "english*.txt" ska köras med "english.txt".
